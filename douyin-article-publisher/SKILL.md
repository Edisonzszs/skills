# 抖音文章自动发布

全自动完成抖音文章的搜索、创作和发布流程。

## 完整流程

```
搜索AI新闻 → 筛选排序 → 提取信息 → 生成文章 → 发布抖音
```

---

## 第一步：搜索AI新闻

从以下科技媒体网站搜索最近24-72小时发布的AI相关新闻：

**权威科技媒体（优先级从高到低）：**
1. TechCrunch
2. The Verge
3. Bloomberg Tech
4. Reuters Tech
5. Wired
6. VentureBeat
7. MIT Technology Review
8. Ars Technica
9. CNBC Tech
10. The Information

**⚠️ 重要原则：**
- 只选择真实媒体报道，不要使用博客或不可靠来源
- 必须验证新闻发布时间（最近72小时内都可以）
- 如果需要国内新闻，搜索时加入"中国"、"OpenClaw"等关键词

**搜索命令：**
```bash
# 国际新闻
node ~/.openclaw/workspace/skills/tavily-search/scripts/search.mjs "AI artificial intelligence OpenAI Anthropic Google Meta Microsoft Nvidia" -n 15 --topic news --days 3

# 国内/OpenClaw 相关新闻
node ~/.openclaw/workspace/skills/tavily-search/scripts/search.mjs "OpenClaw AI 开源 中国 2025" -n 10 --topic news --days 7
```

---

## 第二步：筛选与排序新闻

从所有新闻中选择最重要的 **3-5条**。

**排序原则（按重要性从高到低）：**
1. AI行业重大事件
2. AI公司动态（OpenAI、Anthropic、Google、Meta、Microsoft、Nvidia等）
3. AI政策监管或政府相关（特别是中国政策）
4. AI行业收购、融资、商业合作
5. AI技术突破
6. AI基础设施（芯片、算力、数据中心、电力）

**筛选规则：**
- 按照「行业影响力」从高到低排序
- 如果多个媒体报道同一事件，需要合并信息并列出多个来源
- 每条新闻必须有明确的发布时间和来源
- 国内新闻优先（如果用户指定）

---

## 第三步：提取新闻信息

对于每条新闻 **必须** 提取以下信息：

| 字段 | 说明 |
|------|------|
| 新闻标题 | 完整标题 |
| 发布时间 | 具体日期时间 |
| 新闻来源 | 至少1-2个媒体 |
| 事件核心事实 | 发生了什么 |
| 关键数据 | 如果有 |
| 行业背景 | 相关背景信息 |

## 第四步：生成AI新闻分析文章

### 文章规格
- **总字数：** 1500-2500字
- **风格：** 结构清晰、可读性强、专业但不晦涩

### 标题规范（最重要！）

**标题要求：**
- 最多30个字
- 必须吸引眼球，引发好奇
- 避免平淡的陈述式标题

**推荐标题风格：**

1. **提问式（推荐）**
   - "养龙虾爆了之后，下一个风口是OpenClaw？"
   - "Claude被美国拉黑，为什么反而爆火了？"
   - "AI Agent来了，程序员会被取代吗？"

2. **对比式**
   - "深圳砸72万美元补贴，Nvidia坐不住了"
   - "OpenAI收购、Meta入局，AI安全赛道升温"

3. **数字式**
   - "这3个AI新闻，改变未来10年"
   - "72万美元补贴、概念股暴涨、CCTV点名"

4. **悬念式**
   - "政府砸钱抢人，这个行业要起飞？"
   - "被黑反而爆火，这家AI公司做对了什么？"

**❌ 避免的标题：**
- "今天的AI新闻汇总"
- "AI行业动态速递"
- 平淡的陈述式标题

---

### 摘要规范

**摘要要求：**
- 最多30个字
- 简洁有力，概括核心亮点
- 要能吸引点击

**示例：**
- "深圳砸钱抢人，概念股暴涨"
- "政策+资本+技术，三重驱动"
- "下一个万亿级市场？"

---

### 文章结构（必须严格遵守）

#### 导语（150-200字）
概括核心变化，制造紧迫感，让读者想继续看下去。

**示例：**
> 这周国内AI Agent圈炸了。深圳、无锡政府真金白银砸补贴，最高72万美元。OpenClaw概念股应声暴涨。CCTV点名报道，分析师预测这可能是下一个万亿级市场。国际巨头Nvidia坐不住了，宣布推出竞品。我们来拆解这波狂欢。

---

#### 正文

对于每条新闻使用以下结构：

```
📌 热点一：新闻标题（简短有力）

📅 发布时间：YYYY年MM月DD日
📎 来源：媒体1、媒体2

事件概述（100-150字）
用通俗语言解释发生了什么。

核心内容（150-200字）
• 要点1
• 要点2
• 要点3

深度分析（200-300字）
分析这件事对AI行业的影响，为什么重要。

一句话总结（可选）
用一句话概括这条新闻的意义。
```

---

#### 结尾

总结趋势，给出观点，引发思考。

**固定结尾文案：**
```
🔥 每天3分钟读懂AI世界

#AI #人工智能 #科技新闻 #相关话题标签
```

## 第五步：发布到抖音

使用 OpenClaw Browser 自动化发布：

### 发布流程

1. **打开创作者中心**
   ```
   browser action: open url: https://creator.douyin.com
   ```

2. **点击"我要发文"或"发布文章"**
   - 注意：选择"发布文章"，不是"发布图文"

3. **填写内容**
   - 标题：最多30字
   - 摘要：最多30字
   - 正文：直接粘贴生成的文章

4. **AI配图**
   - 点击"AI配图"按钮
   - 等待生成完成（约5-10秒）

5. **添加音乐**
   - 点击"选择音乐"
   - 搜索激昂类音乐关键词
   - 选择合适的音乐
   - 点击"选择音乐"确认

6. **发布**
   - 确认"公开"和"立即发布"
   - 点击"发布"按钮

### 默认配置

| 配置项 | 默认值 |
|--------|--------|
| 背景音乐 | 激昂类音乐（首选：战马） |
| 文章类型 | 文章（不是图文） |
| 发布时间 | 立即发布 |
| 可见性 | 公开 |
| 发布账号 | openclaw |

## 优化建议

1. **标题优先**：花时间打磨标题，这是决定点击率的关键
2. **热点数量**：3-5个最佳，太少不够丰富，太多读不完
3. **来源标注**：每条新闻都要标注来源，增加可信度
4. **结尾有力**：给出观点或预测，引发讨论
5. **配乐选择**：科技类首选"战马"等激昂音乐，正能量新闻可选"生命力BGM"
