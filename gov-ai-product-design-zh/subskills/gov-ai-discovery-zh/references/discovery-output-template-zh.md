# Discovery 输出模板

建议输出以下结构：

## 1. 场景概述
- 项目名称
- 客户类型
- 牵头部门
- 试点部门
- 核心业务问题

## 2. 当前流程与痛点
- 当前流程
- 人工瓶颈
- 风险点

## 3. 场景分类
- 知识问答 / 文案辅助 / 工作流自动化 / Copilot / Agent
- 分类理由

## 4. AI 必要性判断
- 是否需要 AI
- 理由
- 不用 AI 的备选方案

## 5. Agent 必要性判断
- 是否需要 Agent
- 理由
- 允许的自治范围
- 禁止的自治范围

## 6. 数据与部署边界
- 数据来源
- 系统依赖
- 网络边界
- 部署要求
- 合规要求

## 7. 试点建议
- 试点范围
- 试点对象
- 试点周期
- 成功指标
- 推广前提

## 8. 风险与待确认事项
- 红旗项
- 风险项
- 待确认事项

## 9. 后续建议
- 是否进入 PRD
- 是否进入 AI 或 Agent Spec
- 是否先补调研
