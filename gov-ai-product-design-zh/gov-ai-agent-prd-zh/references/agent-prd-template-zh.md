# 政企 AI 项目 PRD 与 Agent Spec 模板

建议输出拆成三层：

## A. Product Brief

- 项目背景
- 核心问题
- 目标部门与用户
- 业务目标
- 试点范围
- 非试点范围
- 成功标准
- 风险与假设

## B. PRD

- 文档信息
- 背景与目标
- 用户与角色
- 业务流程
- 范围定义
- 功能需求 FR
- 非功能需求 NFR
- 权限模型
- 业务规则
- 页面与交互
- 数据与系统依赖
- 业务验收标准
- 风险与待确认事项

## C. AI 或 Agent Spec

- AI 适配性判断
- 为什么需要 AI
- 为什么需要或不需要 agent
- 智能体角色定义
- 输入来源与知识边界
- 工具清单与调用边界
- 可自动执行动作
- 必须人工确认动作
- 高风险失败模式
- 兜底与转人工策略
- 模型与部署约束
- 日志、审计、追踪要求
- AI 验收指标
- 试点与推广治理机制
