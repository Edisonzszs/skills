# 推荐 MCP 连接

政企 AI 产品规划场景，最常用的 MCP 组合通常是：

- OpenAI Docs MCP：查官方开发文档
- GitHub MCP：PR、issue、仓库协作
- Playwright MCP：浏览器验证、页面流程测试
- Figma MCP：设计稿读取与实现

## OpenAI Docs MCP

官方地址：`https://developers.openai.com/mcp`

适合：

- Codex
- Cursor
- 任何支持 Streamable HTTP MCP 的客户端

## 为什么优先接这几个

- docs：保证引用官方资料
- github：把需求、研发、PR 流程打通
- playwright：验证前端或后台操作流程
- figma：需求到设计到实现闭环

如果是高合规场景，再按需接：

- 企业内部知识库 MCP
- 工单系统 MCP
- 文档管理系统 MCP
- 只读数据库 MCP
