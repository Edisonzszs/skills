# 政企 AI 发现阶段决策门

按顺序过门，不要跳步。

## Gate 1: 是否值得走正式规划

- 是否跨部门或跨角色
- 是否需要后续立项、采购、验收
- 是否存在明显系统接入或合规约束
- 是否需要试点与推广两阶段

如果都不是，可能只需要轻量需求说明，不需要完整 PRD 流程。

## Gate 2: 是否真的需要 AI

- 规则是否已足够稳定且可配置
- 是否只是简单检索或表单流转
- 是否只是已有系统能力缺少串联
- 是否真的存在非结构化信息处理需求

## Gate 3: 是否真的需要 Agent

- 是否需要多步规划与分解
- 是否需要调用多个工具或系统
- 是否需要根据中间结果动态调整步骤
- 是否必须自动做出下一步选择

## Gate 4: 是否有可落地的边界条件

- 数据来源明确
- 网络边界明确
- 部署要求明确
- 权限边界明确
- 审计要求明确

## Gate 5: 是否能做试点

- 是否有清晰试点部门
- 是否有可衡量指标
- 是否有人工兜底
- 是否能控制风险范围
